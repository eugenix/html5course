Что такое HTML5?

HTML* (HyperText Markup Language) - язык гипертекстовой разметки, основная технология Web.

HTML5 - новая версия стандарта HTML, находится в стадии разработки с 2004 года, основной документ www.w3.org/TR/html

HTML5 - "коктель" технологий и стандартов

Немного истории...

  • 2004 - Рабочая группа WHATWG начала работу над новым стандартом
  • 2006 - W3C решает прекратить работу на XHTML и объединить усилия совмество с "WHAT" над HTML5
  • 2008 - Опубликована первая версия
  • Конец 2008 - Броузеры начинают поддерживать стандарт
  • Сентябрь 2011 - 34% Alexa's Top-100 используют HTML5

Немного истории...

  • Ноябрь 2011 - Adobe прекращает поддержку Flash для мобильных устройств
  • Декабрь 2012 - статус Candidate Recommendation для версии 5.0, начата работа над 5.1
  • 2014 - статус Recommendation для 5.0, "фичи" больше не добавляются

Семантика

Семантическая разметка

  • Семантическая разметка упрощает понимание смысла и контекста информации, размещенной на странице, - как для машин, так и для людей
  • Тэги не должны определять отбражение
  • Главная цель тегов - смысловая нагрузка
  • Разрешены перекрывающие тэги

Новые структурные теги

<header> - Определение заголовка страницы или раздела
<footer> - Определение завершителя страницы или раздела
<nav> - Определение области навигации страницы или раздела
<section> - Определение логической области страницы или группировка контента
<articel> - Определение статьи (логически завершенного блока контента)
<aside> - Определение вторичного или связанного контента

Новые структурные теги

<hgroup > - Группировка нескольких заголовочных элементов (h1 - h6) в единый блок
<figure> и <figurecapture > - Вставка дополнительного контента: схем, видео, картинок(с подписью)

Структура документа HTML4

Структура документа HTML5

Пример

                    <!doctype html>
                    <html>
                      <head>
                        <meta charset=utf-8>
                        <title>(Это title) Пример страницы на HTML5</title>
                      </head>
                      <body>
                        <header>
                          <hgroup>
                            <h1>Заголовок "h1" из hgroup</h1>
                            <h2>Заголовок "h2" из hgroup</h2>
                          </hgroup>
                        </header>
                        <nav>
                          <a href=link1.html>Первая ссылка из блока "nav"</a>
                          <a href=link2.html>Вторая ссылка из блока "nav"</a>
                        </nav>

Пример

                        <section>
                          <article>
                            <h1>Заголовок статьи из блока "article"</h1>
                            <p>Текст абзаца статьи из блока "article"</p>
                              <details>
                                <summary>Блок "details", текст тега "summary"</summary>
                                <p>Абзац из блога "details"</p>
                              </details>
                           </article>
                        </section>
                        <footer>
                          <p>Содержимое абзаца из блока "footer"</p>
                        </footer>
                      </body>
                    </html>

Microdata

Микроданные - способ семантически размечать сведения о разнообразных сущностях (событиях, организациях, людях, товарах и так далее)

  • набор пар ключ-значение, добавленных в документ параллельно основному контенту
  • нацелено на автоматизированные парсеры (напр., поисковые роботы)

Пример

                    <section itemscope itemtype="http://data-vocabulary.org/Person">
                      Hello, my name is <span itemprop="name">Vladimir Muzhanovskij</span>
                      I am a <span itemprop="job"> manager </span>
                      from great company <span itemprop="company">System-Plus</span>.
                      My friends call me <span itemprop="nickname">Vovan</span>.
                      You can visit my homepage at <a href="http://www.cuct.ru" itemprop="url">www.cuct.ru</a>.
                      <section itemprop="address" itemscope itemtype="http://data-vocabulary.org/Address">
                        I live at <span itemprop="street-address">2/3 Tramvainaya</span> 
                        <span itemprop="city">Ufa</span>, <span itemprop="region">Russia</span>.
                       </section>
                    </section>
